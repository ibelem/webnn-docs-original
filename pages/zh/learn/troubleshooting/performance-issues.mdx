# Performance Issues

## ONNX Runtime

### How to profile my super slow WebNN implementation?

WebNN EP of ONNX Runtime Web needs specify fixed integer values via [`freeDimensionOverrides`](../tutorials/onnx-runtime/free-dimension-overrides) for all the [symbolic dimensions](../tutorials/onnx-runtime/free-dimension-overrides#symbolic-dimension), oherwise the optimizations will not be applied.

> <InfoIcon /> Oh Wow! fixing the argument names makes it 100x time faster!
