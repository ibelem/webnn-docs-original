# ONNX Runtime

## How to profile my super slow WebNN implementation?

WebNN EP of ONNX Runtime Web needs specify fixed integer values via [`freeDimensionOverrides`](../learn/tutorials/onnx-runtime/freedimensionoverrides) for all the symbolic dimensions, oherwise the optimizations will not be applied.
